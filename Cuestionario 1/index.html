Cuestionario 1: BD vs DBMS

Preguntas a responder:

¿Qué es entonces una base de datos - BD?

Una base de datos es un conjunto de archivos en los que existen múltiples ocurrencias de un mismo tipo de registro, y están interrelacionados entre sí.

¿En qué casos es conveniente usar bases de datos?

Cuando la información que se está manejando necesita ser accesada por varios usuarios o aplicaciones externas, y cuando contiene archivos interrelacionados.

¿Qué es un sistema de gestión de base de datos - DBMS?

Por sus siglas Database Management System, es un software que permite el manejo de colecciones de datos (o bases de datos).
Algunos ejemplos de DBMS incluyen:

  • MySQL
  • SQL Server
  • Oracle

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Administración de Diccionario de Datos
Los DBMS almacenan la información en un diccionario de datos. Los programas accesan a esta información a través del DMBS. [1]

Administración de Almacenamiento de Datos
El DBMS crea y administra estructuras complejas para el almacenamiento de datos. [1]

Transformación y presentación de Datos
El DBMS convierte los datos introducidos en las estructuras de datos requeridas. Le da formato a la información para presentarla de manera adecuada al usuario. [1]

Administración de Seguridad
El DBMS crea un sistema de seguridad que asegura la seguridad del usuario y la privacidad de sus datos. Este sistema de seguridad determina qué usuarios pueden acceder a la base de datos, qué datos puede ver, y qué acciones puede hacer con esos datos ( leer, agregar, modificar, borrar). [1]

Control de Acceso Multi Usuario
El DBMS se asegura de que múltiples usuarios puedan acceder a la base de datos sin comprometer la integridad de los datos. [1]

Administración de Recuperación y Respaldos
En caso de que exista una falla, el DBMS proporciona un respaldo y la recuperación de datos, asegurando así su seguridad e integridad. [1]

Administración de Integridad de Datos
Promueve y emplea reglas de integridad, reduciendo la redundancia, y maximizando la consistencia de la información. [1]

Lenguajes de Acceso a la Base de Datos e Interfaces de aplicaciones programadas.
Proporciona el acceso a la información por medio de un lenguaje de consultas, el cual es un lenguaje no procedimental. La mayoría de los DBMS utilizan SQL (Structured Query Language). [2]

Interfaces de comunicación de Base de Datos
Promueve interfaces intermediarias entre la Base de Datos y el usuario. Por ejemplo, dentro de un navegador web. [2]

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Para la materia de diseño interactivo, mi equipo y yo hicimos una página web sobre la nutrición. Una función que quisimos implementar (pero al final no lo hicimos) era tener un perfil con el nombre, peso, edad y porcentaje de grasa corporal del usuario, para que pudiera ver su progreso al seguir alguna dieta y un plan de ejercicio.
Una base de datos habría sido ideal para implementar esta función, pues tendríamos todos los perfiles de los usuarios, así como las dietas y planes de ejercicio de cada uno.
